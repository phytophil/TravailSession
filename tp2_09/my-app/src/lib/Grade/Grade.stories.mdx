## Description

`Grade` component provides user basic interface to give ratings based on five stars.

`Grade` est une composante de la librairie Materialize-UI pour ajouter une évaluation
sur cinq étoiles. Les versions "modifiable", "lecture seulement", "désactivée" et "sans évaluation"
sont disponibles.
